    1. Add a private method `async _computeContentHash(text)` to SessionOrchestrator:
       ```javascript
       async _computeContentHash(text) {
         const encoder = new TextEncoder();
         const data = encoder.encode(text);
         const hashBuffer = await crypto.subtle.digest('SHA-256', data);
         return Array.from(new Uint8Array(hashBuffer))
           .map(b => b.toString(16).padStart(2, '0')).join('');
       }
       ```
       NOTE: In the test environment, `crypto.subtle` may not be available in jsdom. Use `globalThis.crypto = { subtle: { digest: vi.fn() } }` mock in tests, or use a simple string hash fallback when crypto.subtle is unavailable.

       Add instance property `_reindexInProgress = false` as a guard flag.

    3. Add method `async _indexJournalsForRAG(options = {})`:
       - Get primary journal ID from settings: `game.settings.get(MODULE_ID, 'activeAdventureJournalId')`
       - Get supplementary IDs from settings: `game.settings.get(MODULE_ID, 'supplementaryJournalIds')`
       - Combine into allJournalIds array (primary first, then supplementary).
       - For each journal ID:
         a. Get full text via `this._journalParser.getFullText(journalId)`.
         b. Compute hash via `this._computeContentHash(fullText)`.
         c. Compare with `this._contentHashes[journalId]`. If match, skip (log "Journal [name] unchanged, skipping re-index").
         d. If stale or no prior hash: get chunks via `this._journalParser.getChunksForEmbedding(journalId, { chunkSize: 4800, overlap: 1200 })`.
         e. Convert chunks to RAGDocuments:
            ```javascript
            chunks.map((chunk) => ({
              id: `${journalId}-${chunk.metadata.pageId}-chunk${chunk.metadata.chunkIndex}`,
              title: `${chunk.metadata.journalName} > ${chunk.metadata.pageName} [${chunk.metadata.chunkIndex + 1}/${chunk.metadata.totalChunks}]`,
              content: chunk.text,
              metadata: { ...chunk.metadata, type: 'adventure-journal' }
            }))
            ```
         f. Call `this._ragProvider.indexDocuments(ragDocs, { onProgress: options.onProgress })`.
         g. Store the new hash: `this._contentHashes[journalId] = currentHash`.
       - Return `{ indexed: totalIndexed, skipped: totalSkipped }`.

    4. Call `_indexJournalsForRAG` from `_initializeJournalContext()` (after parsing the journal):

       - Call `_indexJournalsForRAG()` to re-index (only the stale journal will actually be indexed since others' hashes still match).
       - Reset flag. If a queued request exists, process it.
       - This is non-blocking relative to the live session cycle.

         let reindexTimer = null;
         function invalidateJournalCache(journalEntry) {
           const vc = VoxChronicle.getInstance();
           if (vc.journalParser) {
             vc.journalParser.clearAllCache?.();
           }

           // Debounced re-index during live mode only
